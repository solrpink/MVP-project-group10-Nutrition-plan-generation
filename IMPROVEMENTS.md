# Цель:

Внедрение улучшений сервиса на основе обратной связи от тестовой фокус-группы, повышение качества пользовательского опыта, точности генерации плана питания и общей применимости рекомендаций в реальной жизни.

## 1. Улучшения интерфейса и UX

* Формат ответов (убраны JSON-структуры, ответ стал читаемым и форматированным) ✅

* Вынести список покупок в отдельное сообщение ✅

* Вынести отдельными сообщениями планы питания по дням недели✅

## 2. Улучшения логики и качества генерации

* Исправлена работа пищевых ограничений (исключены запрещённые продукты)✅

* Добавлен отдельный блок "предпочтения пользователя" (нелюбимые продукты) ✅

* Добавлены граммовки для каждого ингредиента в блюдах ✅

* Добавлено КБЖУ (калории, белки, жиры, углеводы) для каждого приёма пищи ✅

* Добавлено суммарное КБЖУ за день✅

* Переписан промпт для LLM для улучшения генерации ответа✅

* Исправлен перевод вводимых данных из числового формата в текстовый для качественного понимания вводимых данных для LLM✅

* Добавление еще одного модуля LLM для более широкого ответа на запрос к LLM✅

* Повышена точность соответствия параметрам пользователя✅

* Снижено количество нерелевантных или некорректных рекомендаций✅


## 3. Улучшения анкеты пользователя

* Добавлен параметр пола (м/ж)✅

* Исправлена шкала физической активности (устранено пересечение диапазонов) ✅
