# Telegram-бот для генерации плана питания /  @nutriplaner_bot
### Telegram-бот, который собирает параметры пользователя (цель, возраст, вес, рост, уровень активности, пищевые ограничения) и рассчитывает целевую суточную/недельную калорийность. Далее через API большой языковой модели формируется структурированный план питания в заданном формате, включающий приёмы пищи, калорийность и список покупок. Ответ проходит базовую валидацию (проверка диапазона калорий, структуры данных), после чего форматируется и отправляется пользователю. В архитектуре проекта предусмотрены: обработка пользовательских состояний, работа с API, валидация ответов модели и логирование для последующего тестирования и улучшения сервиса. Ссылка на бота: @nutriplaner_bot

## Функционал
- Сбор параметров пользователя (цель, возраст, вес, рост, активность, ограничения).
- Расчет целевой суточной и недельной калорийности.
- Генерация плана питания через API большой языковой модели.
- Удобный вывод ответа модели пользователю без markdown блоков.
- Валидация ответов модели (проверка калорийности и структуры данных).
- Управление состояниями диалога.
- Логирование событий для тестирования и анализа.

## Технологический стек
- Telegram Bot API
- LLM API (Grok API: LLM Model - llama-3.3-70b-versatile)
- Logging
- Make (автоматизация сборки и задач)
